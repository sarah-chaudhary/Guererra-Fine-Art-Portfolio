Guererra is a WordPress-based artist portfolio built to showcase original artwork collections and limited‑edition prints through a clean, immersive gallery experience.

The website was developed with a custom gallery structure to highlight high‑resolution artwork while maintaining fast performance and responsive design. Visitors can easily explore artwork collections, learn about the artist’s creative journey, and submit inquiries.

Key development work included:
• Custom WordPress portfolio layout for artwork collections
• Optimized image handling for high‑resolution artwork
• Mobile‑responsive gallery experience
• Inquiry system for collectors and potential buyers

The result is a visually elegant portfolio that presents the artist’s work professionally while making it easy for collectors to explore and connect.
